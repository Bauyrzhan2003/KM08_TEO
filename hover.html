CSS Hover эффектісі

Бұл бетте лекция, мысалдар және практикалық тапсырмалар бар.

1. Лекция: CSS hover эффектісі

1.1 hover деген не?

hover – CSS псевдосыныбы, курсор (тінтуір) элементтің үстіне келгенде стильді өзгертуді қамтамасыз етеді.
Пайдаланылады: сілтемелер (<a>), батырмалар (<button>), блоктар (<div>), суреттер (<img>).

1.2 hover қасиеттері

  • Түсін өзгерту (color / background-color)
  • Шекара түсін өзгерту (border-color)
  • Қаріп өлшемін өзгерту (font-size)
  • Трансформациялар (scale, rotate)
  • Транзишн (transition) қосу арқылы жұмсақ ауысу

1.3 Синтаксис

селектор:hover {
    қасиет: мән;
}

1.4 Мысалдар

Мысалы 1: Сілтемеге түсін өзгерту

a {
    color: blue;
    text-decoration: none;
}
a:hover {
    color: red;
}

Мысалы 2: Батырманы үлкейту

button {
    background-color: green;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
    transition: transform 0.3s;
}
button:hover {
    transform: scale(1.1);
}

Мысалы 3: Блокқа фон қосу

div.card {
    width: 200px;
    height: 150px;
    background-color: #f0f0f0;
    border: 2px solid #333;
    transition: background-color 0.3s, border-color 0.3s;
}
div.card:hover {
    background-color: #cceeff;
    border-color: #007BFF;
}

1.5 Практикалық кеңестер

  • Hover тек визуалды өзгеріс үшін қолданылады.
  • Көп қолданылатын қасиеттер: color, background-color, border, box-shadow, transform.
  • transition қасиетімен қосса, өзгеріс жұмсақ болады.
  • Hover мобильді құрылғыларда тек «басқанда» көрінеді, сондықтан тек «тінтуірге арналған» эффекттер үшін.
