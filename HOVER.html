Псевдокласстар арқылы батырмаларды безендіру

1) :hover — Батырманың үстіне курсор келгенде

Теория:
:hover псевдоклассы элементтің үстіне курсор келген кезде стилін өзгертеді.

Код: 
button:hover {
    background: #3498db;
    color: white;
}
Мысалы:
Тапсырма: Өзің 50px ені бар қызыл батырма жасап, hover кезінде жасылға ауыссын.

2) :active — Батырманы басқан сәт

Теория:
:active батырмаға басқан мезетте (клик кезінде) жұмыс істейді.

Код: 
button:active {
    background: #2ecc71;
    transform: scale(0.95);
}
Мысалы:
Тапсырма: Батырмаға басқанда 5px төмен түсетін эффект қос.

3) :focus — Батырма фокус алған кезде

Теория:
:focus батырма пернетақтамен таңдалғанда немесе басылғанда қосылады (TAB арқылы).

Код: 
button:focus {
    outline: 3px solid red;
    background: yellow;
}
Мысалы:
Тапсырма: Фокус түскенде батырма көк түске боялсын.

4) :disabled — Қолдануға болмайтын батырма

Теория:
:disabled күйі батырма бұғатталған кезде қолданылады.

Код: 
button:disabled {
    background: #aaa;
    color: #555;
    cursor: not-allowed;
}
Мысалы:
Тапсырма: disabled батырмаға қызыл рамка қой.
